Chi Siamo

La storia dietro questo progetto educativo

Scopo del progetto

Questo web server è stato creato come progetto educativo per dimostrare come sia possibile realizzare un server HTTP funzionante utilizzando esclusivamente le librerie standard di Python.

L'obiettivo è quello di comprendere i meccanismi fondamentali che stanno alla base del protocollo HTTP e della comunicazione client-server, senza affidarsi a framework complessi.

Tecnologie Utilizzate

  • Python Socket - Per la comunicazione di rete
  • Threading - Per gestire richieste multiple
  • HTML5 & CSS3 - Per l'interfaccia utente
  • JavaScript - Per le animazioni interattive
Python 
# Server HTTP minimale
import socket
import threading

class HTTPServer:
    def __init__(self, port=8080):
        self.port = port
        
    def start(self):
        sock = socket.socket()
        sock.bind(('localhost', self.port))
        sock.listen(5)
        
        while True:
            client, addr = sock.accept()
            self.handle_request(client)
